# OAPS HTTP Examples

These examples document the current canonical HTTP binding surface.

- `discovery-contract.v1.json` maps every normative endpoint to request/response examples
- `*.response.json` files show canonical response shapes for the current reference slice
- `requests/` contains representative mutation bodies
- `errors/` contains stable OAPS error payload examples reused across the binding draft
- `media-type.*.json` captures the canonical `application/oaps+json` target and the draft compatibility `application/json` allowance
