# OAPS Foundation Schemas

These schemas are the first machine-readable contracts for the suite-level core foundation draft.

They are intentionally narrower than the total long-term OAPS vision.

Current foundation object set:

- actor
- capability
- intent
- task
- delegation
- mandate
- approval-request
- approval-decision
- challenge
- execution-result
- evidence-event
- error-object
- extension-descriptor
- interaction-transition
- task-transition

These schemas should evolve with `spec/core/FOUNDATION-DRAFT.md` rather than with the older consolidated draft spec.
