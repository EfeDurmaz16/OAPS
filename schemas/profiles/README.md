# OAPS Profile Draft Schemas

This directory holds machine-readable schemas for cross-profile draft helper
objects that appear across auth, trust, payment, provisioning, and compatibility
declaration examples.

Current entry points:

- `subject-binding-assertion.json`
- `trust-attestation.json`
- `payment-challenge.json`
- `provisioning-operation.json`
- `profile-support-declaration.json`
