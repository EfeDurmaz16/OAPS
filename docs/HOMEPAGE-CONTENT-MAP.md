# AICP Homepage Content Map

## Purpose

This page organizes the public-facing homepage narrative so the landing page can stay concise while still pointing to the right deeper documents.

## Page Rule

The homepage should feel like an invitation into a layered protocol suite, not like a product brochure or a monolithic platform pitch.
